## 使用方法

将 TOC 项目传递给组件。

```mdx
import { InlineTOC } from 'fumadocs-ui/components/inline-toc';

<InlineTOC items={toc} />
```

### 在页面中使用

您可以在每个页面中添加内联目录。

```tsx
<DocsPage>
  ...
  <InlineTOC items={toc} />
  ...
</DocsPage>
```
