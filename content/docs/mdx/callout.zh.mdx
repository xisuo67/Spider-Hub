## 使用方法

将其添加到您的 MDX 组件中。

```tsx
import { Callout } from 'fumadocs-ui/components/callout';

<MDX
  components={{
    Callout,
  }}
/>;
```

有关用法，请参见 [Markdown](/docs/markdown#callouts)。
